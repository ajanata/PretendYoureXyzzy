Pretend You're Xyzzy game

Card Database

This Pretend You're Xyzzy server stores its card data in a PostgreSQL database. The script to create this database and populate it with the standard Cards Against Humanity cards is licensed under a Creative Commons - Attribution - Noncommercial - Share Alike license, just like Cards Against Humanity itself.
